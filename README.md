# Campus Cart

Campus Cart is a small Amazon-style e-commerce project built with React, Bootstrap, Node, Express, and a JSON file database.

## Run the app

Install dependencies:

npm install

Build the React app:

npm run build

Start the web server:

npm start

Open:

http://localhost:3000

## Features

- Home page with product cards already loaded
- Bootstrap navbar with search bar and cart link
- Carousel/jumbotron section
- Best sellers section
- Product detail pages when clicking product images or names
- Add to cart from the home page and product detail page
- Cart page with saved items, quantity controls, and trash buttons
- Category filters for product browsing
- Product detail pages with an in-stock label
- Cart summary with subtotal, tax, shipping, and estimated total
- Checkout confirmation
- Reset cart button for testing
- About and Contact pages
- JSON database persistence in `server/db.json`
- External product and carousel images loaded from stock image URLs
